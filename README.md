# FilmBoxd

<i><sup>Note: This README file is bound to change with edits as the project grows</sup></i>

A personal project building a web application inspired by LetterBoxd, where users can track, rate and review movies.

## Tech Stack

- Frontend

  - ReactJS
  - TailwindCSS
  - MantineUI (For certain components)

- Backend

  - NodeJS with ExpressJS
  - PostgreSQL

- API
  - Will be using `TMDB`'s API for fetching information related to the movie itself.
